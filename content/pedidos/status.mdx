# Status e Rastreio

Cada pedido passa por uma sequência de status. O cliente pode acompanhar o andamento em tempo real pela vitrine.

## Ciclo de status

| Status | Quem define | O que significa |
|---|---|---|
| **Pendente** | Sistema | Pedido recebido, aguardando aceite |
| **Em preparo** | Você | Pedido aceito, cozinha produzindo |
| **Saiu para entrega** | Você | Produto saiu com o entregador (delivery) |
| **Pronto para retirada** | Você | Produto está pronto no balcão (takeaway) |
| **Concluído** | Você | Entrega confirmada ou retirada realizada |
| **Cancelado** | Você | Pedido recusado ou cancelado |

## Notificações para o cliente

O cliente pode receber e-mail nos seguintes eventos (se informou e-mail no checkout):

| Evento | E-mail enviado? |
|---|---|
| Pedido confirmado | ✅ Sim |
| Saiu para entrega | ✅ Sim |
| Pedido concluído | ✅ Sim |
| Pedido cancelado | ❌ Não (notifique manualmente) |

<Callout type="info">
  O e-mail é enviado automaticamente se o cliente informou o endereço de e-mail no checkout.
</Callout>

## Rastreio na vitrine

Após finalizar o pedido, o cliente recebe um link de acompanhamento. Para ver o status mais recente, o cliente precisa **recarregar a página** — o link fica disponível enquanto o pedido não for arquivado.
