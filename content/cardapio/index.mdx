# Cardápio

O cardápio é organizado em **categorias** e **itens**. Cada item pode ter grupos de **opcionais** (adicionais, variações, tamanhos). Tudo configurado pelo painel.

## Estrutura

```
Cardápio
├── Categoria (ex.: Pizzas)
│   ├── Item (ex.: Calabresa)
│   │   ├── Grupo de opcionais (ex.: Tamanho)
│   │   │   ├── Broto — R$ 29,90
│   │   │   └── Grande — R$ 49,90
│   │   └── Grupo de opcionais (ex.: Borda)
│   │       ├── Sem borda
│   │       └── Cheddar — R$ 5,00
│   └── Item (ex.: Frango com Catupiry)
└── Categoria (ex.: Bebidas)
```
